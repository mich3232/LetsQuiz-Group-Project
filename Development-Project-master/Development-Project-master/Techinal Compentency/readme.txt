Technical Competency Demo
This application aims to serve as a demonstration that the development team of Let's Quiz has the technical ability to deliver the product based on their proposed architecture

Prerequisites
- Unity3D Editor
- If using .apk file: Android device running OS version of 5.0 or later that is connected to the internet.
- If using .exe file: Desktop computer connected to the internet

Deployment
This application serves only as demonstration and as such is limited to basic functionality of retrieving, inserting, and deleting data from a database. This database is hosted at 000webhost on a free service plan that stops every day from 00:00-01:00 AEST. 

Built With
- Unity3D
- C#
- PHP
- MySQL Database
